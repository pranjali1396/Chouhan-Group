How to Integrate CRM with Your Vercel Website

Step 1: Add Script to Your Website

Copy the CRM integration script to your Vercel project's public folder.

Step 2: Include Script in Your HTML

Add this before the closing </body> tag in your contact page:

<script src="/crm-webhook-integration.js"></script>

Step 3: Update API URL

In crm-webhook-integration.js, change:

API_URL: 'http://localhost:5000/api/v1/webhooks/lead'

To your production backend URL:

API_URL: 'https://your-backend-api.railway.app/api/v1/webhooks/lead'

Step 4: Verify Form Field Names

Make sure your form uses these field names (or update the script):

  • name or Name - Customer name
  • phone or Phone - Phone number
  • email or Email - Email address
  • broker - Are you a broker? (Yes/No)
  • source - How did you hear about us?
  • homeType - Home type interested in?

Step 5: Test

  1. Start your backend: cd backend && npm run dev
  2. Submit a test form on your website
  3. Check backend console - you should see the lead data

Example Form HTML (Your Current Form Structure)

<form id="contactForm">
  <input type="text" name="name" placeholder="Name" required />
  <input type="email" name="email" placeholder="Email" required />
  <input type="tel" name="phone" placeholder="Phone" required />
  <select name="broker">
    <option value="No">No</option>
    <option value="Yes">Yes</option>
  </select>
  <select name="source">
    <option value="Google Search">Google Search</option>
    <option value="Social Media">Social Media</option>
    <!-- etc -->
  </select>
  <select name="homeType">
    <option value="Flat">Flat</option>
    <option value="Bungalow">Bungalow</option>
    <!-- etc -->
  </select>
  <button type="submit">Submit</button>
</form>

<!-- Add this before </body> -->
<script src="/crm-webhook-integration.js"></script>

For Next.js (If Your Site Uses Next.js)

Add to pages/_app.js or app/layout.js:

import Script from 'next/script';

export default function MyApp({ Component, pageProps }) {
  return (
    <>
      <Component {...pageProps} />
      <Script 
        src="/crm-webhook-integration.js" 
        strategy="afterInteractive" 
      />
    </>
  );
}
